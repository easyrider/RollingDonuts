# RollingDonuts
妖精さんがドーナツと一緒に転がるゲーム。

### 概要
Unityでつくるカジュアルゲーム。
iPhone6plusで縦持ちで遊べる予定です。

とあるgamejamに参加して事前に行ったアイデアソンでのネタをベースに製作。

### アイデアベース
人間が落としたドーナツに強襲される妖精さんのピンチをチャンスに変える物語。
ドーナツに轢かれそうになりながらドタバタで転がるドーナツの上に乗ることに成功した妖精さん。
転がっている先はなんと偶然にも仲間のいる住処だった！
このままうまいこと振り落とされずに帰れたら・・・？

仲間達と美味しくドーナツを頂こうとちょっと邪な思考に至った妖精さんががんばるゲーム。
転がるドーナツの上に乗っている妖精さんが落ちないように操作して、ゴールまでドーナツを運ぼう。
